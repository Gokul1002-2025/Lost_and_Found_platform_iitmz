# ThatzMine

An AI-powered Lost and Found platform that helps people reconnect with their lost belongings using semantic search and image matching.

## Features

- Report lost items
- Report found items
- AI-powered matching
- Image upload
- User authentication
- Search and filters
- Admin dashboard

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- FastAPI

### Database
- PostgreSQL

### AI
- Sentence Transformers
- CLIP (later)
